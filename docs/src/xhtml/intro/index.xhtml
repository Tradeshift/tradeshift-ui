UI components range from simple buttons and icons to fancy datepickers and dropdown menus. The target audience for Tradeshift UI is both developers and designers.

For Developers

For Tradeshift apps, the UI components are built to allow for consistent styling and experiences across internal apps. For third-parties, the UI components are designed to bring visual consistency with internal applications as part of the Tradeshift platform.
